## CV Format Best Practices:

**Structure:**
1. Header (Name, Title, Contact)
2. Education
3. Research Interests
4. Publications/Research
5. Professional Experience
6. Skills
7. Projects
8. Certifications
9. Awards & Achievements
10. References (optional)

**Design:**
- Use professional fonts (Arial, Calibri, Garamond)
- Keep it 1-2 pages (for BBA students)
- Use bullet points for readability
- Include white space
- Use consistent formatting
- Proofread multiple times!
